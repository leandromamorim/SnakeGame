# SnakeGame

<br>This is a simple Snake Game.</br>
<br><b>Game Objectives:</b></br>
Catch much food as possible without crashing with the snake body or wall.
<br></br>
<b>Used:</b>
<br>Java, IntelliJ IDEA</br>
<br><b>Controls:</b></br>
UP, DOWN, LEFT , RIGHT
